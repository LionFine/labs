# Лабораторная работа №3
## Вариант 16. Фильтрация изображений и морфологические операции

Реализована морфологическая дилатация со структурирующим элементом `disk 3x3`:
```text
0 1 0
1 1 1
0 1 0
```

Подготовка входных данных:
- перевод в полутон по взвешенному среднему;
- бинаризация Bradley-Roth (`5x5`, `t = 15%`);
- для бинарной морфологии foreground-интерпретация задаётся белыми пикселями текста.
